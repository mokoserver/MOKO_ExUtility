# MOKO_ExUtility

Проект, предоставляющий внешние утилиты для платформы MOKO SE.

## Описание

Этот репозиторий содержит набор вспомогательных утилит, которые могут быть вызваны из основного скрипта MOKO SE. Утилиты реализованы как на Python, так и на LabVIEW.

- **CSMInfo.py**: Python-скрипт, который предоставляет утилиту для ввода и получения подробной информации об образце и условиях испытаний (климатические условия, настройки оборудования, параметры теста). Это позволяет централизованно управлять вводом данных перед началом тестирования.
- **TLM.vi**: Утилита на LabVIEW, назначение которой не до конца ясно из названия, но, вероятно, связана с телеметрией или управлением.

Вызов утилит из Python-скриптов осуществляется через функцию `MOKO.Utility()`.

## Структура проекта

- **PythonDocs/**: Папка с Python-реализациями утилит.
  - `CSMInfo.py`: Утилита для ввода информации об испытаниях.
- **Utulities/**: Папка с утилитами на LabVIEW.
  - `TLM/TLM.vi`: Утилита TLM.
- **MOKO exUtility.lvproj**: Файл проекта LabVIEW для объединения и управления утилитами.

## Ключевые технологии

- Python
- LabVIEW
